The quiet empty state for a pane or scope: faint `icon`, semibold `title`, one-or-two-line `body`, optional `action` button label.
Keep the copy calm and instructive; no illustrations, no exclamation marks (brand rule).

```jsx
<SLEmpty icon="tray" title="Nothing needs attention" body="Failed or unpublished meetings would appear here." />
```
