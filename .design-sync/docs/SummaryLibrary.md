The Library window gallery: the full three-pane meetings browser (scope rail, meeting list, detail with Summary / Transcript / Audio tabs) plus cropped states like batch selection, empty library, and processing rows.
No props; it renders a gallery of the shipped Library states. For focused mockups prefer its parts: SLStatusPill, SLSegmented, SLSection, SLBullets, SLActionItem, SLEmpty.
