A tinted status pill for permission-like states: `tone` picks the color (granted = success, needed = warning, denied = danger, neutral = subtle), with an `icon` name and `text`.

```jsx
<PWStatusPill tone="granted" icon="check-circle" text="Granted" />
<PWStatusPill tone="needed" icon="alert" text="Needs access" />
```
