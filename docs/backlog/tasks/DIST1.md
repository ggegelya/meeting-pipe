# DIST1: Bundle a runtime for a drag-n-drop installer

Band origin: carried from Q4; base spec in q4-final.md, progress notes here. Status and priority live in this task's ToC row in [meetingpipe-q6-backlog.md](../meetingpipe-q6-backlog.md).

**DIST1 (P2, partial): bundle a runtime for a drag-n-drop installer.** Base spec in [`q4-final.md`](./q4-final.md) (search the id). **Progress (2026-07-12): daemon plumbing + build tool + runbook shipped; the notarized bundle build + clean-Mac test are owner-owed.** Design + runbook: [`docs/spikes/dist1-bundle-runtime.md`](../spikes/dist1-bundle-runtime.md). The problem is that a clean Mac has no Python 3.11+ and the install needs Homebrew + uv + ffmpeg (and locked-down regulated Macs often cannot install Homebrew at all, the exact zero-egress target). Approach: embed a relocatable python-build-standalone (`install_only`, no baked rpaths) + the pipeline wheels + a static ffmpeg under `Contents/Resources/pipeline-runtime/`. **Shipped + tested:** `PipelineLauncher.findMP` now prefers the embedded runtime and runs it as `python3 -m mp` (NOT the `bin/mp` console script, whose pip-baked absolute shebang breaks when the .app is relocated to /Applications, the crux relocation gotcha), extracted into a pure injected `resolveDirectMP` with 4 `PipelineLauncherFindMPTests` pinning the priority (embedded > venv > uv > PATH); `MeetingRecorder.findFFmpeg` falls back to `pipeline-runtime/bin/ffmpeg` after PATH, covering the recorder merge and `MuteRedactor`. The build tool `scripts/bundle-runtime.sh` fetches the runtime, `pip install`s the pipeline into it, verifies `python3 -m mp --version`, installs a static ffmpeg (owner pins an LGPL/GPL-compliant `FFMPEG_URL`), and signs every embedded mach-O inside-out (a notarization prerequisite). 1224 daemon green. **Reconciliation:** DEP1 (GO on native AVFoundation merge/transcode) does not remove ffmpeg because `MuteRedactor` is a third spawn site out of DEP1's scope, so a static ffmpeg is still bundled (the `findFFmpeg` fallback makes dropping it later a no-op); I7 (drop Python) is the strategic fork DIST2 recommended deferring, and bundling now buys a shippable clean-Mac install without betting on the port. Owner-owed remainder (needs Developer ID creds + a second clean Mac, neither here): run `scripts/bundle-runtime.sh` with a real `CODESIGN_IDENTITY` + `FFMPEG_URL`, notarize (`notarytool submit --wait`) + staple, package a .dmg, and drag-install-test on a clean Mac confirming (via `events.jsonl`) the pipeline ran entirely from the bundle. D8 is the durable CI home for the notarize step once a second user exists.
